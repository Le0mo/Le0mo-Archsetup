# Le0mo-Archsetup
安装shorin niri

```bash
bash <(curl -L shorin.xyz/archsetup)
```
进入niri后
```bash
sudo pacman -S clang
niri-blur-toggle
```
克隆仓库
```bash
mkdir Github
cd Github
git clone https://github.com/Le0mo/Le0mo-Archsetup.git
```
更换配置
```bash
chmod +x ~/Github/Le0mo-Archsetup/new.sh
bash ~/Github/Le0mo-Archsetup/new.sh
```
